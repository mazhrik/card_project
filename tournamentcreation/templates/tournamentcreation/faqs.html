FAQs for BBO Teams Application

Question: What is the purpose of this application?
Answer: This application is designed to generate a file which can be sent to BBO to start multiple team matches at the same time
Question: What is the tournament name?
Answer: It is a name that describes your tournament. Try to keep it short and meaningful. E.g “USBF Mixed Trials”, “D22 GNT Finals”.
Question: What is a Session?
Anwser: A session is a group of matches where the same teams are playing against each other. This could be a Round in a Swiss game or a Knockout. A session may have more than one segment if the boards played will be split and played at different times (normally not applicable to Swiss format).
Question: How do I name my sessions?
Anwser: Give them names that are meaningful and descriptive. E.g. “Day 1 Swiss Round 3”, “Knockouts Semifinal”
Question: When should I submit usernames to BBO for validation?
Anwser: Usernames should be submitted to BBO for validation no later than 3 days before the start of the tournament.
Question: What should I do if there is a last minute lineup change?
Anwser: If there is a last minute lineup change, make the changes on the appropriate page(s) and regenerate the TeamMatch file and resubmit to BBO.
Question: Can I add players to a team after the tournament starts?
Anwser: The system will allow you to add players at any time. Be sure to check the Conditions of contest to be sure it is permitted.
Question: Why are there two places to assign TDs?
Anwser: TDs can be assigned “globally” (meaning they are assigned to all matches) or they can be assigned “locally” (meaning they are just assigned to certain tables).
Question: What is the barometer option?
Anwser: The barometer option determines whether players see the results during the match (Option set to “Yes”) or only at the end of the match (Option set to “No”). It is most common to leave the barometer set to “N” to prevent a team from changing strategies based on a known advantage or disadvantage in previously played boards.
Question: What is the undo option?
Anwser: The undo option allows the players at the table to request an undo for a bid or play. Play will stop and the opponents must approve or deny the undo request. If this is set to “No”, then players cannot request an undo and would need to call a TD for assistance.
Question: Who/what is the host?
Anwser: The BBO host/username is the creator of the tournament and will be the host of all matches. This userid must have sufficient rights to create the appropriate number of matches. If you are not sure about the host and its permissions, contact your BBO coordinator.
Question: Can I use the same predealt hands in all the matches instead of random hands?
Anwser: You may use predealt hands which will be used in each match. To add them, you will need to check the box in the Matches screen and add them to the generated Match Creation file after the description and before the first Match. The format is Board [tab] number=x [tab] deal =SxxxxHxxDxxxCxxx, SxxHxxxDxxxCxxx etc for all four hands starting with the South hand and ending with a Carriage Return. The boards are simply used in the order they appear. Board numbers will start from 1 unless the Starting board number setting in the Matches screen is used to specify a different starting board number.
